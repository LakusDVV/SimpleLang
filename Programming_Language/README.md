# SimpleLang — Мини-язык для математики (C# 7.3)

Простой интерпретируемый язык для целых чисел.  
Все переменные объявляются через `let`.  

---

## Ключевые слова

| Ключ | Действие |
|------|----------|
| `let` | объявление переменной (`let x = 5`) |
| `input()` | ввод числа из консоли |
| `output(expr)` | вывод значения выражения |

---

## Примеры

### Объявление и операции
let x = 5
let y = x + 3
output(y)



### Степень
let z = x ^ 2
output(z)



### Ввод и вывод
let a = input()
output(a + 10)



---

## Поддерживаемые операции

| Операция | Пример |
|----------|--------|
| `+` | a + b |
| `-` | a - b |
| `*` | a * b |
| `/` | a / b |
| `^` | a ^ b |

---

## Запуск

Запуск программы
При старте программы можно выбрать один из вариантов:

Ввести путь к исходному файлу вручную.

Использовать файл из корневой папки проекта (где находится .exe).

Запустить demo, если файл не выбран.


Примечание: «корневая папка проекта» — это папка, где лежит .exe после сборки (например, bin/Debug/net7.0/).



Если путь не указан — запускается встроенный пример (demo).
